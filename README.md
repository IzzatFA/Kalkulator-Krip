# CryptoGlass | Web Cipher Calculator

CryptoGlass is a sleek, modern, web-based cipher calculator built with Python (Flask) and a glassmorphism aesthetic. It supports encryption and decryption using 5 classic cryptographic algorithms.

## Features
- **Modern UI**: Animated gradients and frosted "glass" panels.
- **5 Ciphers Implemented**: Vigenere, Affine, Playfair, Hill, Enigma.
- **Responsive Design**: Works nicely on both desktop and mobile screens.

## Installation & Usage

1. **Clone the repository**:
   git clone https://github.com/IzzatFA/Kalkulator-Krip.git
   cd Kalkulator-Krip

2. **Install requirements**:
   pip install -r requirements.txt

3. **Run the application**:
   python app.py

4. **Access the Web Interface**:
   Open http://127.0.0.1:5000 in your browser.
